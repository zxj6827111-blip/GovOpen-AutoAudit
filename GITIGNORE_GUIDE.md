# =============================================================================
# .gitignore 配置说明
# =============================================================================
# 
# 重要提示：
# .env 文件包含敏感信息（API密钥），不应提交到Git仓库
# 
# 使用方法：
# 1. 确保项目根目录有 .gitignore 文件
# 2. 在 .gitignore 中添加以下内容：
#
# # 环境变量配置（包含敏感信息）
# .env
# 
# # Python
# __pycache__/
# *.py[cod]
# *$py.class
# *.so
# .Python
# 
# # 运行数据
# runs/
# *.log
# 
# # IDE
# .vscode/
# .idea/
# *.swp
# *.swo
# 
# # OS
# .DS_Store
# Thumbs.db
# 
# # Playwright
# .playwright/

# 如需提交模板，使用 .env.example (不包含真实密钥)
